# Examples

Example scripts demonstrating PhysicsNeMo Curator pipelines.

## Basics

| Example | Description |
|---------|-------------|
| [00_pipeline](_basics/00_pipeline/) | Build a Source → Filter → Sink pipeline |
| [01_parallel](_basics/01_parallel/) | Run pipelines in parallel with `run_pipeline` |
| [02_checkpoint](_basics/02_checkpoint/) | Checkpoint and resume pipeline execution |
| [03_dashboard](_basics/03_dashboard/) | Launch the interactive metrics dashboard |

| Example | Description |
